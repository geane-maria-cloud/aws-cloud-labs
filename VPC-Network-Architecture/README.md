
# Projeto AWS - Arquitetura de Rede com VPC

## Descrição

Este projeto demonstra a criação de uma arquitetura básica de rede na AWS utilizando VPC, sub-redes públicas e privadas.

## Tecnologias utilizadas

- AWS VPC
- Subnets
- Internet Gateway
- Route Tables
- Security Groups
- EC2
- Apache
- MariaDB

## Etapas realizadas

1. Criação de uma VPC personalizada
2. Criação de sub-rede pública
3. Criação de sub-rede privada
4. Configuração de Internet Gateway
5. Configuração de tabelas de roteamento
6. Configuração de Security Groups
7. Provisionamento de instância EC2
8. Deploy de aplicação web

## Objetivo

Praticar conceitos de arquitetura de rede, segurança e infraestrutura em nuvem utilizando AWS.

## Resultado

Ambiente de rede funcional com controle de acesso e aplicação web executando em instância EC2.
