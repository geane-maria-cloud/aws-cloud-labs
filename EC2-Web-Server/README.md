# Projeto AWS - Servidor Web com EC2

## Descrição
Este projeto demonstra o provisionamento de um servidor web utilizando Amazon EC2 em ambiente de laboratório.

## Tecnologias utilizadas

- AWS EC2
- Amazon Linux
- Apache HTTP Server
- Security Groups

## Etapas realizadas

1. Criação de instância EC2 (t3.micro)
2. Configuração de Security Groups (SSH e HTTP)
3. Instalação do Apache HTTP Server
4. Deploy de página web estática
5. Validação de acesso via IP público

## Objetivo

Praticar conceitos fundamentais de provisionamento de infraestrutura em nuvem utilizando AWS.

## Resultado

Servidor web funcionando e acessível publicamente via navegador.
